Skalenauswahl

Bitte wählen sie für das Kriterium eine Skala aus, mit welcher das Kritierum gemessen werden soll. Die gewählte Skala bestimmt die Auswertungsmöglichkeiten für das jeweilige Kriterium.

Nominalskala

Skala, deren Ausprägungen keine Ordnung besitzen, aber unterscheidbar sind.

Geschlecht
männlich, weiblich
Orte
Köln, Berlin, München

Ordinalskala

Erstellen Sie eine Skala, deren Werte geordnet und unterscheidbar sind.

Zensuren
sehr gut < gut < befriedigend < ausreichend < mangelhaft < ungenügend
Zufriedenheit
sehr zufrieden < zufrieden < unzufrieden < sehr unzufrieden

Metrische Skala / Kardinalskala

Abstände der Werte dieser Skala sind messbar, jedoch besitzt diese Skala keinen natürlichen Nullpunkt.

Temperatur
Grad Celsius; Grad Fahrenheit; Kelvin
Zeitdauer
Jahre; Tage
