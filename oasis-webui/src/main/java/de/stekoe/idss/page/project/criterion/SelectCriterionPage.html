Skalenauswahl

Bitte whälen sie für das Kriterium eine Skala aus, mit welcher das Kritierum gemessen werden soll. Die gewählte Skala bestimmt die Auswertungsmöglichkeiten für das jeweilige Kriterium.

Referenzskalen

Wählen Sie aus einer Vielzahl an vorgefertigten Skalen und passen Sie sich diese an Ihre Bedürfnisse an!

Nominalskala

Skala, deren Ausprägungen keine Ordnung besitzen, aber unterscheidbar sind.

Orte

Ordinalskala

Erstellen Sie eine Skala, deren Werte geordnet und unterscheidbar sind.

Zensuren

Zufriedenheit
